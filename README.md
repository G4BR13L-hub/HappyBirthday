Un Codelab desarrollado con Jetpack Compose que muestra una tarjeta de cumpleaños. Este proyecto demuestra el uso de componentes declarativos mediante funciones componibles (@Composable). La interfaz utiliza una Column con disposición centrada y textos personalizados con tamaños en scale-independent pixels (sp). Incluye sistema de temas con Material Design 3 y previsualizaciones interactivas en Android Studio para agilizar el desarrollo.
